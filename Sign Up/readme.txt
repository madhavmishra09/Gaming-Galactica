# Sign Up Page

This page allows new users to register for Gaming Galactica.

## Features

- Collects username, email, and password.
- Validates input fields (required, email format, password match).
- Redirects to the Login page after successful registration.
- Displays error messages for invalid input.

## Usage

1. Fill in the registration form.
2. Submit to create a new account.
3. On success, you will be redirected to the Login page.

## Notes

- User data management (storage, password hashing) must be implemented separately.
- Users are not logged in automatically after registration.
- For any issues, ensure form validation and error handling are in place.
